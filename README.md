# construction-python-tools

A suite of open-source Python utilities designed to streamline construction workflows, automate repetitive data tasks, and improve project delivery efficiency.

## 🚀 Key Features
*   **Material Takeoffs:** Automated calculation tools for concrete volume, steel reinforcement weight, and framing counts.
*   **Schedule Analysis:** Scripts to parse Primavera P6 or MS Project exports to identify critical path delays.
*   **Cost Estimation:** Utilities to map material quantities to unit-price databases.
